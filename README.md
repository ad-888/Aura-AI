# Aura-AI

Assistant personnel IA en français, propulsé par Claude (Anthropic). PWA installable, fonctionne directement dans le navigateur.

## Configuration

1. Obtenez une clé API Anthropic sur [console.anthropic.com/account/keys](https://console.anthropic.com/account/keys)
2. Ouvrez l'application et cliquez sur l'icône **⚙** en haut à droite
3. Collez votre clé (format `sk-ant-…`) et cliquez sur **Enregistrer**

La clé est stockée uniquement dans le `localStorage` de votre navigateur — elle ne transite jamais par un serveur tiers.

## Fonctionnalités

- Quatre modes : Général, Professionnel, Créatif, Planification
- Actions rapides (email, résumé, brainstorming, correction…)
- Contexte personnel (École Black Jack, Loge Parfaite Amitié)
- PWA installable, cache hors-ligne via Service Worker
